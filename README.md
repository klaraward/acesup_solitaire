# Aces Up

Ett klassiskt patiensspel byggt med vanilla JavaScript.
Vibe-kodat med Claude Code.

**Spela nu:** https://klaraward.github.io/acesup_solitaire/

## Regler

1. Klicka på kortleken för att dela ut fyra kort
2. Klicka på ett kort för att ta bort det om ett annat kort med samma färg (hjärter, ruter, klöver, spader) har högre värde
3. Ess är högst och kan aldrig tas bort
4. Flytta kort till tomma platser genom att klicka på kortet och sedan den tomma platsen
5. Målet är att bara ha de fyra essen kvar
